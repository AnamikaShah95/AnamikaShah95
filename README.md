## 😀 Face Recognition using PCA + ANN

> **A Machine Learning project implementing face recognition using Principal Component Analysis (Eigenfaces) and Artificial Neural Networks.**

### ✨ Overview

This project demonstrates how dimensionality reduction and neural networks can be combined to build an efficient face recognition system. It focuses on preprocessing facial datasets, extracting Eigenfaces using PCA, and classifying identities with an Artificial Neural Network.

### 🚀 Features

- 📷 Face Detection
- 😀 Face Recognition
- 🧠 PCA (Eigenfaces)
- 🤖 ANN Classification
- 📊 Image Processing Pipeline
- 📈 Model Evaluation
- 🗂 Dataset Preprocessing
